# Airline-Loyalty-Analysis-Tableau
Airline loyalty and customer behavior analysis using Tableau dashboards

# Tools Used

Tableau Desktop — Dashboard design & analysis
CSV Dataset — Airline loyalty data
Data Dictionary — Attribute definitions

# Dashboards Overview

Dashboard screenshots are available in folder:
dashboard-screenshots/

# 1. Airline Membership Overview

Key KPIs:
Total Members
Net Members
Cancelled Members
Cancellation Percentage

Insights include:
Membership trends over time
Cancellation vs total member comparison
Flight activity by enrollment type
CLV differences by marital status
Education-level distribution of members

# 2. Promotion-Based Airline Membership

This dashboard focuses on promotion vs standard enrollment behavior.

Key insights:
Promoted members show different flight activity patterns
CLV varies significantly across marital status and gender
Province-level flight activity reveals geographic differences
Loyalty card tiers strongly influence CLV and points velocity

# Key Business Insights

Cancellation rate (~11.8%) highlights the importance of retention strategies
Promotional enrollments show distinct engagement and flight patterns
Married customers tend to have higher average CLV
Higher loyalty tiers (Star) correlate with higher CLV and engagement
Geographic variation suggests region-specific marketing opportunities

# How to Use

Download the .twbx file and open it in Tableau Desktop
Explore filters such as enrollment type, province, education, and city
Use dashboard screenshots for a quick overview without Tableau
